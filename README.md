# Proyecto Base

Ingresar desde la terminal del proyecto en visual

Instalaciones:

Backend:

```bash
cd backend
npm install
```

Frontend:

```bash
cd frontend
npm install
```

Ejecutar el Backend:

```bash
cd backend
npm run dev
```

Ejecutar el Frontend:

```bash
cd frontend
npm run dev
```

Ingresar a la direccion proporcionada por el frontend
